# AI Vault Indexer for Obsidian

This plugin automatically generates a comprehensive index of your Obsidian vault, specifically formatted for **AI agents** and **LLMs** (Large Language Models). 

## Why this plugin?
Most AI tools struggle to understand the full structure of a deep folder hierarchy. This plugin creates a single, clean document that maps out your entire vault, using clear chapter separators (`---` and titles) to ensure an AI can parse your infrastructure without getting lost.

## Features
- **Automatic Indexing:** Generates a structured map of all your files and folders.
- **AI-Ready Format:** Uses specific line-based chapter separation for better token recognition by AI models.
- **Infrastructure Clarity:** Helps LLMs understand where information is located before they start reading individual notes.

## Installation
- Jus extract the .zip into obsidian plugins folder
*(Note: Once officially approved, it will be available in the Obsidian Community Plugin store).*

## How to use
Simply run the command `Generate Vault Index` from the Command Palette (`Ctrl/Cmd + P`), and a new index file will be created/updated in your root folder.
